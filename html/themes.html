Themes
splot allows user to select from a series of themes that will affect colors and other visual options.

When changing the theme from the Preferences panel in the application is necessary to restart the application for the new selected theme to be applied.

The feature allows also to define custom themes.

Theme search path: splot will look for themes in the themes directory where splot is installed. Additional directories where splot will look for themes can be configured through the environment variable SPLOT_THEMES. The SPLOT_THEMES variable can contain one or several directories separated by a :. For example:

  export SPLOT_THEMES=~/mystuff/themes:/opt/themes
